# 🔐 Secret Message Web App

A simple authentication-based web application built using **Node.js** and **Express.js**.
Users can enter a password to unlock and view a secret message.

---

## 🚀 Features

* 🔑 Password-based access system
* 📄 Serves static HTML pages
* ⚡ Built with Express middleware
* 🔒 Simple authentication logic

---

## 🛠️ Tech Stack

* **Node.js**
* **Express.js**
* **Body-parser**
* **HTML (Frontend)**

---

```
project/

```

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/Secret_Message_Proj.git
```

2. Navigate to the project folder:

```
cd Secret_Message_Proj
```

3. Install dependencies:

```
npm install
```

4. Run the server:

```
node index.js
```

5. Open in browser:

```
http://localhost:3000
```

---

## 🔐 How It Works

* User enters a password on the home page.
* Middleware checks the password.
* If correct → Secret page is displayed.
* If incorrect → User stays on login page.
